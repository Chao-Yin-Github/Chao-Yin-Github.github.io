  1. 先 clone 仓库 (前提是 gitbook 仓库)

  1. 安装 npm 和相关 module 以及其他插件

    # 安装 npm
sudo pacman -S npm

# 安装 gitbook 命令行工具
sudo npm install gitbook-cli -g

# 此时 gitbook -V 可以看到版本号

# 安装caliber
sudo -v && wget -nv -O- https://download.calibre-ebook.com/linux-installer.sh | sudo sh /dev/stdin

# 检查所需要的 ebook-convert 插件是否安装成功
ebook-convert --version

    CALIBER 全平台下载地址

  2. 准备生成 EBOOK

    cd [项目文件夹]
# 可以看到有 SUMMARY.md,cover.jpg 等,确认是一个 gitbook类型的仓库

# 安装所需插件
gitbook install 

# 生成电子书以 pdf 为例
gitbook pdf

# 可以看到有一个 book.pdf 文件已经生成
